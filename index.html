# UMM..WHAT IS THIS?


A very basic and lightweight CSS framework built for developers. You love Bootstrap's grid but you waste a lot of time to remove or modify all of the other classes every time you create a new template? EVRIKA! This one's for you!

And...what's so special?

You're right, there are a lot out there. But we REALLY do know what you're looking for. A fast lightweight framework packed with essential starter elements ONLY : a basic grid, basic buttons, basic menu, all responsive, Font Awesome integration, Google Font integration and the rest is on you! But even with this basic elements, this page still looks clean & modern, right?

Is it FREE?

Yes, of course! This is our way of giving something back to the design & development community. But if you want to buy us a coffee, we would appreciate it a lot!

Ok, ok, how do I start?

Download this zip file to get WOW CSS Framework. Included files: CSS, HTML, JS. You will have two starter templates, one that looks almost like this demo and another one, blank. So...get ready to build your awesome template and come back to show us your work! :)
